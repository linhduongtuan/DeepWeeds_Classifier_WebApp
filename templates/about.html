About this project

This project is aimed to help Pytorch machine learning developers to quickly build a Flask web app in a Docker container ready to be deployed.
View on GitHub

By default, this app uses EfficientNet_Lite4 to train 9 classes of weeds collected in Australia.
View dataset, paper and baseline code on GitHub

This can be easily changed with any custom deep learning model. If you wish to contribute to the project, please feel free to add more example models. More explanation as to how to contribute can be dound in the GitHub repository link.
